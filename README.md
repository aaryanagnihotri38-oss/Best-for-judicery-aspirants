# ⚖️ Law and Order by Aaryan Agnihotri

A premium educational website for Judiciary Aspirants focused on mastering the Indian Penal Code (IPC) through structured learning, landmark cases, interactive revision plans, and exam-oriented preparation.

## 🌟 Features

### 🎯 Hero Section
- Premium modern UI
- Responsive design
- Animated statistics
- Call-to-action buttons

### 📚 IPC Learning Modules
- General Principles (Sections 1–106)
- Offences Against Human Body
- Offences Against Property
- Public Order Offences

### 🔥 High Priority Topics
- Sections 299–304 (Culpable Homicide vs Murder)
- Sections 34 & 149 (Common Intention vs Common Object)
- Sections 375–376D (Rape & Sexual Offences)

### ⚖️ Landmark Cases
Includes important Supreme Court judgments:

- Virsa Singh v. State of Punjab
- K.M. Nanavati v. State of Maharashtra
- Govinda v. State
- Mahbub Shah v. Emperor
- Shambhu Nath Singh v. State
- Barendra Kumar Ghosh v. Emperor

### 📅 30-Day Elite Revision Plan
Interactive study tracker with:

- 30-day schedule
- Progress saving
- Completion tracking
- Revision checkpoints

### 🧠 Interactive Quiz
Features:

- Multiple-choice questions
- Instant feedback
- Score tracking
- Restart functionality

### 🎨 Premium Design
- Dark luxury theme
- Gold accents
- Glassmorphism effects
- Smooth animations
- Floating particles
